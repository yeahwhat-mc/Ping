Ping
====

This plugin does nothing but pinging the `/ping`command issuer or a specific player.


## Permissions

Permission | Purpose | Default
--- | --- | ---
`ping.self` | Ping yourself using `/ping` | true
`ping.others` | Ping other players using `/ping <player>` | ops
